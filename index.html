Pasos para subir un Cambio a Git

		
		Inicializar el Repositorio:

		- git init


		Agregar los cambios:

		- git add . (Sube todas la modificaciones)

		- git add index.html


		Ver Estado de los Archivos:

		- git status


		Crear el commit (Confirmaci√≥n del cambio)

		git commit -m  "add index.html"
		

		- Subir Cambio 
		git push -u origin master


		- Agregar un Origen del Repositorio
		git remote add origin https://github.com/claudito/tutorial-git-2.git
